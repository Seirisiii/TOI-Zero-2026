## เกณฑ์การผ่านการฝึกเขียนโปรแกรมบนแพลตฟอร์ม TOI National Grader

### นักเรียนต้องทำโจทย์เขียนโปรแกรมบนแพลตฟอร์ม TOI National Grader และมีผลการทดสอบผ่านเกณฑ์ ดังนี้

1. ทำโจทย์ระดับ A2 และ A3 รวมกันอย่างน้อย 20 ข้อ
2. ทำโจทย์ทุกระดับ (A1, A2 และ A3) อย่างน้อย 40 ข้อ
3. นักเรียนต้องได้รับคะแนนไม่น้อยกว่า 80 คะแนน จากคะแนนเต็ม 100 คะแนน ในแต่ละข้อที่ทำ

ทั้งนี้ สำหรับนักเรียนที่เคยเข้าร่วมโครงการ TOI-Zero ในปีที่ผ่านมา มูลนิธิฯ จะมีการเพิ่มเติมโจทย์ใหม่ในบางส่วน และจะคัดโจทย์บางข้อออกจากการนับผลคะแนนในปีนี้ โดยไม่ได้ลบโจทย์เก่าออกจากระบบ

นักเรียนยังคงสามารถทำโจทย์เก่าได้ตามปกติ แต่ผลคะแนนการทำโจทย์ในข้อที่ถูกคัดออก จะไม่ถูกนำมาคิดรวมในการพิจารณาการผ่านเกณฑ์ โครงการ TOI-Zero ประจำปีการศึกษา 2569

การคัดเลือกโจทย์บางส่วนไว้ในระบบ จะช่วยให้นักเรียนเก่าที่เคยทำมาแล้ว ยังคงได้รับประโยชน์บางส่วนจากผลงานเดิม พร้อมทั้งเป็นแรงจูงใจให้ฝึกทำโจทย์ใหม่เพิ่มเติม ในขนะที่นักเรียนใหม่ก็จะได้ทำโจทย์ในชัดเดียวกันกับนักเรียนทุกคน จึงถึงเป็นแนวทางที่เป็นธรรมและเท่าเทียม ต่อนักเรียนผู้เข้าร่วมใหม่และนักเรียนผู้ที่เคยเข้าร่วมโครงการมาแล้ว โจทย์ข้อที่ไม่ถูกนำมาพิจารณาคะแนน จะประกาศให้ทราบเมื่อมีการเปิดใช้งานระบบ TOI National Grader

### ระดับความยากของโจทย์ แบ่งออกเป็น 3 ระดับ ดังนี้

- ระดับ A1 : การรับค่า-แสดงผล การใช้ตัวแปร และโครงสร้างควบคุมพื้นฐาน (if-else, loops)
- ระดับ A2 : การจัดการข้อมูลพื้นฐาน เช่น การประมวลผลค่าที่รับเข้ามา การใช้ฟังก์ชัน และการจัดรูปแบบข้อมูล
- ระดับ A3 : การใช้เงื่อนไขซับซ้อน การวนลูปซ้อน และการคำนวณค่าตามเงื่อนไขที่ซับซ้อนขึ้น

<ins>หมายเหตุ</ins> : โจทย์ทั้งหมดอยู่ในขอบเขตของการเขียนโปรแกรมเบื้องต้น ตามเนื้อหาหลักสูตรวิทยาการคำนวณ ไม่มีเนื้อหาเกี่ยวกับ algorithm และ data structure ที่ซับซ้อน
